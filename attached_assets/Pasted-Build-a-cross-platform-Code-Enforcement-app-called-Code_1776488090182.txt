Build a cross-platform Code Enforcement app called "Code Enforcement Hub" using Expo / React Native with web support.

The app must work for:
- iOS
- Android
- Web/desktop browser

Main purpose:
A case management app for code enforcement that allows an inspector or office staff member to:
- create and manage cases
- assign case numbers automatically
- store property and responsible party information
- add one or more violations to a case
- link each violation to an ordinance section
- store ordinance text in a searchable ordinance library
- upload and view photos for each case
- write inspection notes
- track notice stages
- generate notice letters
- preview notice letters in-app
- export notices as PDF
- track case status from open to closed

Design goals:
- clean professional municipal style
- simple mobile-first layout for field use
- responsive web layout for desktop office use
- easy navigation
- no AI detection features
- no map integration yet
- no payment features yet

Use a local mock database or structured local storage first, but organize the code so it can later connect to a real backend.
